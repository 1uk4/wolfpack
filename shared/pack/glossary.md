# Pack Glossary

| Term | Meaning |
|------|---------|
| wolf | An always-on Claude Code agent on a dedicated host |
| den | A wolf's private Obsidian-compatible brain (`workspace/den/`) |
| shared | Read-only pack library pushed from the Mac hub (`workspace/shared/`) |
| pack | The collection of all wolves + the Mac hub |
| 1uk4 | Owner/operator of the wolfpack |
| hub | 1uk4's Mac — the central Syncthing node that all wolves connect to |
